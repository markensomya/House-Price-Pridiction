# House Price Prediction

A machine learning project that predicts house prices based on property features.

## Dataset
- Source: Kaggle — Housing Prices Dataset
- 545 rows, 13 columns
- Target: price

## Features Used
area, bedrooms, bathrooms, stories, parking, mainroad, guestroom, basement, hotwaterheating, airconditioning, prefarea, furnishingstatus

## Project Structure
- analysis.ipynb — Main Jupyter Notebook (all 5 tasks)
- Housing.csv — Dataset
- Summary.pdf — Written summary of findings
- charts/ — Folder with all 3 chart images

## Models Used
- Linear Regression — R² Score: 0.65
- Random Forest Regressor — R² Score: 0.61

## Key Finding
Area (square footage) is the strongest predictor of house price, followed by bathrooms and air conditioning.

## Tools
Python, Jupyter Notebook, Pandas, Scikit-learn, Matplotlib, Seaborn
